Homework 4: Linked Lists

calendar_today Due Oct 15, 2026 timer 3 days remaining

description Implementation Requirements

In this assignment, you will implement a doubly linked list data structure from scratch. You are required to handle all edge cases, including operations on empty lists and single-node lists.

Core Methods to Implement:

  • insert_at_head(data): O(1) complexity
  • insert_at_tail(data): O(1) complexity
  • remove_node(node_ptr): Proper memory management required
  • reverse_list(): In-place reversal of pointers

Memory leaks will be checked using Valgrind. Ensure that your destructor correctly deallocates all heap-allocated nodes. Submit your source code as a single linked_list.cpp file or a compressed archive if you use custom headers.
